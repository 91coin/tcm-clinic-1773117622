## 🚀 快速开始

### 首次使用
1. 打开系统页面
2. 系统自动初始化数据（20 个患者/15 个方剂/50 种药材）
3. 开始使用各功能模块

### 完整就诊流程
```
1. 患者管理 → 新建档案
   ↓
2. 预约接诊 → 现场挂号
   ↓
3. 中医诊疗 → 接诊开方
   ↓
4. 处方管理 → 查看处方
   ↓
5. 收费结算 → 确认收费
   ↓
6. 药房库存 → 发药扣减库存
```

## 💾 数据备份

### 导出数据
1. 进入"系统管理"页面
2. 点击"📤 导出数据"按钮
3. 下载 JSON 格式备份文件

### 导入数据
```javascript
// 在浏览器控制台执行
const data = prompt('请粘贴备份的 JSON 数据');
importSystemData(data);
```

### 自动备份
系统每次操作都会自动保存到 localStorage，无需手动备份。
