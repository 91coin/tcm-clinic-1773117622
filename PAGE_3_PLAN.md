# Phase 3 开发计划 - 中医诊疗开方

## 开发内容

### 1. 写病历功能
- [ ] 望闻问切四诊合参
- [ ] 辨证论治
- [ ] 诊断和证型选择
- [ ] 治法填写

### 2. 开处方功能
- [ ] 选择方剂（从经典方剂库）
- [ ] 自定义加减
- [ ] 药材剂量设置
- [ ] 用法说明
- [ ] 处方金额计算

### 3. 方剂库功能
- [ ] 方剂列表展示
- [ ] 方剂详情查看
- [ ] 方剂检索
- [ ] 方剂调用开方

### 4. 药材库功能
- [ ] 药材列表展示
- [ ] 药材详情查看
- [ ] 药材检索
- [ ] 药材剂量参考
